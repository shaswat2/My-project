# SCM 2.0 Project

**A learning project built with Java and AWS for understanding cloud deployment and backend development.**

---

## Project Overview
This project is created for learning purposes. It simulates a **Supply Chain Management (SCM) system** to manage resources, orders, and tracking. The focus was on learning **Java backend development**, **Spring Boot**, and **AWS deployment**.

---

## Features
- User management and authentication
- Resource and inventory tracking
- Order creation and management
- Deployed backend APIs on AWS

---

## Tech Stack
- **Backend:** Java, Spring Boot  
- **Database:** MySQL / H2  
- **Deployment:** AWS EC2 / AWS RDS  
- **Build Tools:** Maven / Gradle  
- **Version Control:** GitHub
